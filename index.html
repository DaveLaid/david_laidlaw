Bookmark Buddy

Bookmark Buddy is the last website you will ever need to remember when referencing your bookmarks.

Bookmark Buddy was built in a week using the MERN stack for the purpose of creating a useful bookmarking website while honing in skills in Mongo, Express, React, and Node. This website allows a user to create an account, login, and save their favorite websites. Not only does Bookmark Buddy save the urls, you can also add a title, category, note and screenshot of the website section you want to remember. Bookmark Buddy is the last website you will ever need to remember when referencing your bookmarks!

  • Website: https://bookmark-buddy.herokuapp.com
  • Date: September 2017
  • Client: Clark Nielsen
  • Category: Website

Mongo Scraper

Mongoose & Cherrio focused web app that scrapes recent gaming articles from OpenSource.com.

Mongo Scraper is a web app utilizing MongoDB, Mongoose, Node, Express, and Cherrio to scrape recent gaming articles from OpenSource.com. You can also store your favorite articles, as well as make notes/comments about these articles to refer to at a later time.

LIRI Bot

LIRI is a Language Interpretation and Recognition Interface.

LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data. LIRI is capable of scraping Twitter account data, Spotify data, and even data from OMDB's API.

  • Website: https://github.com/DaveLaid/liri-node-app
  • Date: August 2017
  • Client: UCLA
  • Category: Node App

Bamazon

Node App designed for split Manager / Customer views of inventory/prices/stock.

Bamazon is an Amazon-like storefront built with Node.js & MySQL. The app will take in orders from customers and deplete stock from the store's inventory. The app is also programed to track product sales across each department and also provides a summary of the highest-grossing departments in the store.
